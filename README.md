# Aeroporturi principale din România

Acest proiect oferă o vizualizare web a aeroportului principal din România (Henri Coandă), utilizat ca sursă în evaluarea strategică a zgomotului conform Directivei 2002/49/CE.

## Vizualizare hartă

👉 https://zgomot.github.io/aeroport-principal/

## Tip serviciu

- Serviciu: WMS (Web Map Service)
- Standard: OGC WMS 1.1.1
- Sursă: GeoNoise MMAP

## Descriere

Layer-ul afișează aeroportul principal (MajorAirportSource) care reprezintă modul de identificare a aeroportului principal.

## Instituție responsabilă

Ministerul Mediului, Apelor și Pădurilor (MMAP)

## Observații

Aceasta este o vizualizare web pentru explorarea datelor. Pentru utilizare oficială, consultați serviciile INSPIRE, website MMAP sau descărcați fișierul gpkg din www.data.gov.ro .
